# Учет АКБ и ТСД в логистическом центре

Описание: на терминале работает приложение, которое складывает инфу в БД. Происходит периодический опрос системы, терминалы на Windows CE. Задача для диплома я думаю только просмотр/анализ/редактирование информации в этой БД. Т.е. как появляется информация в этой БД останется за кадром или написать обобщенно. Терминал я с работы все равно не заберу, да и демонстировать это онлайн не получится, поэтому берем автономную БД, которую я прикрепил. Поэтому думаю нужно делать просмотр/анализ/редактирование информации в этой БД, автономно. Т.е. разместить БД или в самом проекте или как базу на ПК. В БД можно добавить побольше каких-нибудь придуманных данных, на основе существующих. По идее для регистрации в системе нового терминала устанавливается клиентская часть на терминал или он подцепляется. Но в целом для диплома указать что регистрирует новое устройство - системный администратор. Также нужно добавить в функционал контроль тсд в ремонте. Указать неисправность, дату отправки в ремонт, можно стоимость (могу предоставить реальную таблицу с ценами по ремонту, она небольшая). Для отдела АХО, например, можно сделать доступ к ограниченному функционалу, например только прсмотр каких-то данных без редактирования.
